# Data

The small sample dataset shipped in `raw/population_africa_sample.csv` is an
illustrative snapshot of 2023 African population and GDP-per-capita figures.
Values are rounded and meant strictly as a teaching artifact — do NOT cite
them in real research.

## Real data sources

For real analysis, download from:

- [World Bank Open Data](https://data.worldbank.org/) — CC-BY-4.0
- [UN Data](https://data.un.org/) — public domain
- [Africa Open Data](https://africaopendata.org/) — depending on dataset
- [data.gouv.bj](https://data.gouv.bj/), [DataFirst SA](https://www.datafirst.uct.ac.za/)
